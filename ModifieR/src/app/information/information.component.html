Welcome to ModifieR
A web based tool for inference of disease modules.
ModifieR is a web based tool, allowing for inferrence of disease modules using multiple algorithms. It takes a list of genes as input, allows you to choose a PPI networ to use for the analyss, and outputs a list of genes included in the disease module.
If you need help getting started, have a look at our prefilled example!
